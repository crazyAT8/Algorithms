Bubble Sort

Bubble sort is a straightforward sorting algorithm that repeatedly steps through the list,
compares adjacent elements, and swaps them if they are in the wrong order.
This process is repeated until the list is sorted.
The algorithm gets its name bc smaller elements "bubble" to the top of the list (beginning of the array) with each iteration.

// pseudo code

procedure bubbleSort( A : list of sortable items )
    n := length(A)
    repeat
        swapped := false
        for i := 1 to n-1 inclusive do
            if A[i-1] > A[i] then
                swap(A[i-1], A[i])
                swapped := true
            end if
        end for
        n := n - 1
    until not swapped
end procedure
  1. initialize 'n': Set 'n' to the length of the list 'A'
  2. Repeat loop: The algorithm repeats the following steps until no swaps are made in a complete pass (indicated by 'swapped' being 'false').
    • Set "swapped" flag: At the beginning of each pass, set "swapped" to "false".
    • Iterate over the array: For each element in the array (from the first element up to the second-to-last element):
      • Compare and swap: if the surrent element ('A[i-1]') is greater than the next element ('A[i]'), swap them and set 'swapped' to 'true'.
    • Reduce 'n': After each pass, reduce 'n' by 1 because the last element is correctly positioned and doesn't need to be compared again.
  3. Until condition: The loop continues until a complete pass is made without any swaps ("swapped" is "false"), indicating that the list is sorted.

This pseudo code provides a clear, language-agnostic representation of the bubble sort algorithm, highlighting the logic and steps involved without the syntax specifics of any programming language.


// JavaScript

function bubbleSort(arr) {
    let n = arr.length;
    let swapped;
    do {
        swapped = false;
        for (let i = 1; i < n; i++) {
            if (arr[i - 1] > arr[i]) {
                // Swap elements
                let temp = arr[i - 1];
                arr[i - 1] = arr[i];
                arr[i] = temp;

                swapped = true;
            }
        }
        // Reduce n because the last element is already in place
        n--;
    } while (swapped);
    return arr;
}

// Example usage:
const arr = [64, 34, 25, 12, 22, 11, 90];
console.log("Sorted array:", bubbleSort(arr));
  • This function takes an array as input and sorts it in ascending order.
  • It uses a 'do-while' loop to keep running until no more swaps are needed
  • indicating that the array is sorted
  • the 'swapped' flag is used to track whether any elements were swapped in the surrent iteration
  • if no elements were swapped, the array is sorted, and the algorithm can terminate.
